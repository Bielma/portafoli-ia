## Getting Started

Either clone the repo with 

```sh
git clone https://github.com/bielma/portafolio-ia.git
```

Or use the Gatsby CLI

```sh
gatsby new material-album https://github.com/bielma/portafolio-ia.git
```

then simply run

```sh
yarn
```

to install dependencies
